Overview

This dataset covers 39 countries across six regions that experienced significant regime changes from the 1990s through the mid-2020s. For each country, 20 indicators across four dimensions are scored annually on a 0–100 scale, where higher values represent greater stability, openness, and state capacity.

Data collection spans approximately 32 years per country (coverage rate: 99.8%). All scores are stored as YAML files and aggregated programmatically; no values are hardcoded into the dashboard.

Country Selection

Countries were selected to cover the full spectrum of regime change outcomes:

  • Violent/unstable — ongoing fragmentation, civil war, or collapse (e.g. Syria, DRC, South Sudan)
  • Violent then recovery — serious conflict followed by stabilization (e.g. Sierra Leone, Liberia)
  • Peaceful then backsliding — democratic opening reversed (e.g. Tunisia, Egypt, Georgia)
  • Managed/partial — elite-led transitions with limited democratization (e.g. Algeria)
  • Peaceful successful — durable democratic transitions (e.g. Ghana, Indonesia, Croatia)
  • Electoral then authoritarian — competitive elections leading to autocracy (e.g. Venezuela)

Six regions are represented: Middle East & North Africa, Africa (violent), Africa (peaceful), Eastern Europe/Former Soviet, Asia, and Latin America.

Scoring Scale

All indicators use a 0–100 scale where 100 represents the best possible outcome and 0 represents the worst. The scale direction is consistent: a higher score always means a better situation, regardless of whether the underlying data is "good when high" (e.g. GDP) or "good when low" (e.g. inflation, violence). Inversions are applied during data collection.

Quantitative indicators are transformed to the 0–100 scale using documented formulas (e.g. (8 - FH_civil_liberties) / 7 * 100). Qualitative indicators use controlled vocabulary feature sets with corresponding score anchors; assessors select the feature that best describes the observed situation.

Dimension Scores

Each dimension score is the weighted average of its five indicators. All indicator weights default to 1.0 (equal weight). Missing data is handled by skip_and_renormalize: the average is computed over available indicators only, with weights adjusted proportionally.

One exception: Extractive Transparency is excluded from the Transparency dimension for 13 countries where the indicator is not applicable (no significant extractive sector).

Composite Score

The composite score is the weighted average of the four dimension scores, with equal weights (1.0 each). As with indicators, missing dimensions are handled by skip_and_renormalize.

The composite score is the primary metric shown by default in the dashboard. Users can select individual dimensions or indicators, or build custom weighted combinations using the Custom Metric Builder.

Source Reliability Tiers

Each data point is assigned a source reliability tier:

  • High — peer-reviewed academic publications, established international indices (V-Dem, Freedom House, World Bank)
  • Medium — reputable think-tank reports (ICG, IISS), major news organizations, government statistics
  • Low — single-source reports, contested figures, extrapolations

Data status is classified as complete (full data available), partial (incomplete but usable), or unavailable (no usable data). Unavailable years are excluded from dimension score calculations.

Dashboard Features

The interactive dashboard supports:

  • Overlay mode — all selected countries on a single chart for direct comparison
  • Stacked mode — one subplot per country with shared, linked x-axis (pan/zoom synchronizes)
  • Absolute x-axis — calendar years
  • Aligned x-axis — years relative to each country's first regime change (t=0 at regime change)
  • Pivot x-axis — same as aligned but the reference year is user-controlled per country via slider
  • Custom metrics — weighted combinations of any indicators, computed on the fly
  • Shareable URLs — full dashboard state (countries, metrics, mode, time range) encoded in URL hash

Analysis Overlays

Four optional overlays are available from the toolbar. All default to off. All computations are client-side from the underlying score data.

A. Confidence Bands

Shaded regions showing per-year scoring uncertainty. Two methods:

  • Data quality — band width from source_type + data_status: quantitative+complete → ±5 pts; qualitative+complete → ±10 pts; partial → ±18 pts; unavailable → no band.
  • Score variance — ±1σ of a 3-point centered window: σ = std({y[i−1], y[i], y[i+1]}). Requires ≥2 neighbors.

B. Volatility Layer

Trailing rolling standard deviation over a 3- or 5-year window: σ_t = std(y[t−w+1], …, y[t]). Wide bands = noisy series; narrow bands = stable or trending series.

C. Trend Indicators

Arrow annotation at end of each series. Two methods, over last 5 years:

  • Linear slope — OLS regression β (pts/yr). ↑ if β ≥ +0.5, ↓ if β ≤ −0.5, → otherwise.
  • Mann-Kendall τ — non-parametric monotone trend: τ = (P−Q) / (n(n−1)/2) where P/Q = concordant/discordant pairs. ↑ if τ > 0.3, ↓ if τ < −0.3, → otherwise. Robust to outliers.

D. Source-change Markers

Diamond markers (◇) at years where source_type transitions between qualitative and quantitative. A score change coinciding with a source-type change may reflect a measurement artifact, not a real-world change. Only meaningful for single atomic indicators (not composite/dimension metrics).

Limitations

  • Qualitative scoring subjectivity — assessors must choose between feature categories; different assessors may score ambiguous situations differently.
  • Composite score masking — equal-weight averaging can obscure important within-country variation. Use individual indicator views for detailed analysis.
  • Coverage gaps — some indicators are unavailable for specific country-years, particularly for transparency indicators in conflict-affected states. The 0.2% non-coverage is concentrated in a small number of country-years.
  • Comparability across regimes — a score of 50 for political violence in Syria means something different from 50 in Ghana. The scale is designed for within-country temporal comparison, not cross-country ranking.
  • Data currency — data extends through approximately 2025–2026 for most countries but may lag for rapidly evolving situations.
