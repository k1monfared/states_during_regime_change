# Plot Regime Change Trajectory Data

Create visualizations comparing countries' post-regime-change trajectories.

## The Plotting Script

All plots are generated by `data/scripts/plot_data.py`. Your job is to interpret the user's request and call this script with the right arguments.

## Script Usage

```bash
python3 data/scripts/plot_data.py [OPTIONS]
```

### Selection (pick one)
- `--countries iraq,libya,tunisia` — comma-separated country IDs
- `--region mena` — all countries in a region
- `--category violent_unstable` — all countries in a category

### What to Plot
- `--dimension composite` — (default) overall composite score
- `--dimension political|economic|international|transparency` — specific dimension score
- `--indicator territorial_control` — specific indicator (requires --dimension)
- `--show-dimensions` — all 4 dimensions on one plot (single country only)
- `--show-indicators` — all indicators within a dimension (single country only)

### Plot Style
- `--overlay` — all countries on same plot (default: separate subplots)
- `--align-regime-change` — set regime change year as x=0
- `--plot-type line|heatmap` — visualization type

### Output
- `--output plots/myplot.png` — save to file
- `--dpi 150` — resolution for saved image
- (no --output) — display interactively

### Discovery
- `--list-regions` — show available regions
- `--list-categories` — show available categories

## Available Regions

- `mena` — Middle East & North Africa
- `africa_violent` — Africa (violent transitions)
- `africa_peaceful` — Africa (peaceful transitions)
- `eastern_europe` — Former Soviet / Eastern Europe
- `asia` — Asia
- `latin_america` — Latin America

## Available Categories

- `violent_unstable` — violent, prolonged instability
- `violent_then_recovery` — violent, then successful recovery
- `peaceful_successful` — peaceful, successful transition
- `peaceful_then_backsliding` — peaceful then democratic backsliding
- (and others — run `--list-categories` to see all)

## Example Commands

### Compare MENA countries (overlay)
```bash
python3 data/scripts/plot_data.py --region mena --overlay --output plots/mena_composite.png
```

### Iraq all dimensions over time
```bash
python3 data/scripts/plot_data.py --countries iraq --show-dimensions --output plots/iraq_dimensions.png
```

### Peaceful African transitions aligned to regime change
```bash
python3 data/scripts/plot_data.py --region africa_peaceful --overlay --align-regime-change --output plots/africa_peaceful_aligned.png
```

### Political dimension heatmap for violent transitions
```bash
python3 data/scripts/plot_data.py --category violent_unstable --dimension political --plot-type heatmap --output plots/violent_political_heatmap.png
```

### Tunisia political indicators
```bash
python3 data/scripts/plot_data.py --countries tunisia --dimension political --show-indicators --output plots/tunisia_political_indicators.png
```

### Compare specific countries on specific indicator
```bash
python3 data/scripts/plot_data.py --countries iraq,syria,libya --dimension political --indicator territorial_control --overlay --output plots/territorial_control_comparison.png
```

## Interpreting User Requests

| User says | Translate to |
|-----------|-------------|
| "compare Iraq and Libya" | `--countries iraq,libya --overlay` |
| "show me MENA countries" | `--region mena` |
| "all peaceful transitions" | `--category peaceful_successful` |
| "political stability over time" | `--dimension political` |
| "how does territorial control compare" | `--dimension political --indicator territorial_control` |
| "aligned to regime change" | `--align-regime-change` |
| "one plot for each" | (default, no --overlay) |
| "overlay them" / "same plot" | `--overlay` |
| "heatmap" | `--plot-type heatmap` |
| "save it" | `--output plots/<descriptive_name>.png` |

## Prerequisites

The script requires:
- `matplotlib` (`pip install matplotlib`)
- `numpy` (for heatmaps: `pip install numpy`)
- Score data must exist (run `python3 data/scripts/generate_scores.py` first)

- Overlay or separate plots?
